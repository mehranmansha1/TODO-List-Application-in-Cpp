# Console-Based TODO List Application (C++)

A simple and efficient console-based TODO List Application developed in **C++** that allows users to manage daily tasks using **CSV file storage**.

---

## 📌 Features

- ➕ Add New Task
- 📋 View All Tasks
- 🔍 Search Task by ID
- ✏ Update Existing Task
- ❌ Delete Task
- ✅ Mark Task as Completed
- 📂 View Completed Tasks
- 📂 View Pending Tasks
- 💾 CSV File Storage

---
